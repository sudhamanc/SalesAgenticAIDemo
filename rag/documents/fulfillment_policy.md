## Installation Scheduling

### Scheduling Process

#### Step 1: Initial Scheduling (Within 24 hours of order approval)
- Customer contacted by scheduling team
- 3 installation date options provided
- Installation window: 8 AM - 6 PM
- 4-hour appointment windows available

#### Step 2: Pre-Installation Confirmation (48 hours before)
- Confirmation call/email to customer
- Site readiness checklist reviewed
- Special requirements confirmed
- Technician assigned

#### Step 3: Day-Before Reminder
- Automated reminder sent
- Technician contact information provided
- Estimated arrival time (2-hour window)

#### Step 4: Day-Of Communication
- Technician calls 30 minutes before arrival
- Customer notified of any delays
- Real-time tracking available via customer portal

### Installation Windows
- **Standard**: Monday-Friday, 8 AM - 6 PM
- **Extended**: Monday-Friday, 7 AM - 8 PM (+$100)
- **Weekend**: Saturday, 9 AM - 5 PM (+$200)
- **After-Hours**: Weekdays after 6 PM (+$150)

### Scheduling Priority
1. Service outages and repairs
2. Expedited installations (paid)
3. Premium SLA customers
4. Standard installations (FIFO)

## Installation Procedures

### Pre-Installation Site Survey

**Required For:**
- Managed WiFi installations
- Complex network setups
- Multi-floor installations
- New construction

**Survey Checklist:**
- [ ] Verify physical access to installation location
- [ ] Identify fiber entry point
- [ ] Measure cable runs
- [ ] Identify power outlet locations
- [ ] Check for interference sources (WiFi)
- [ ] Photograph installation areas
- [ ] Obtain building/landlord approvals
- [ ] Identify any special requirements

### Standard Internet Installation Procedure

**Duration**: 2-4 hours

**Step 1: Outside Plant (30-60 minutes)**
1. Locate fiber drop at property line
2. Verify fiber connectivity
3. Run fiber to building entry point
4. Install fiber through conduit/wall penetration
5. Seal entry point
6. Test fiber signal strength

**Step 2: Inside Plant (60-90 minutes)**
1. Run fiber to equipment location
2. Install ONT (Optical Network Terminal)
3. Connect ONT to power
4. Install business router
5. Configure router with customer network settings
6. Connect customer equipment for testing

**Step 3: Testing and Validation (30-60 minutes)**
1. Verify fiber link status
2. Test internet connectivity
3. Run speed test (must meet or exceed subscribed speed)
4. Test static IP configuration (if applicable)
5. Verify router security settings
6. Document installation details

**Step 4: Customer Training (15-30 minutes)**
1. Demonstrate router admin interface
2. Provide WiFi credentials (if applicable)
3. Review customer portal access
4. Explain support contact methods
5. Obtain customer signature on completion form

### Voice Service Installation Procedure

**Duration**: 1-2 hours (per 5 phones)

**Step 1: Network Preparation (15-30 minutes)**
1. Verify internet service active
2. Configure QoS (Quality of Service) on router
3. Set up VLAN for voice traffic (if required)
4. Test network connectivity

**Step 2: Phone Deployment (30-60 minutes)**
1. Unbox and inventory phones
2. Connect phones to network
3. Power on phones
4. Verify auto-provisioning
5. Test dial tone

**Step 3: Configuration (30-45 minutes)**
1. Assign phone numbers to extensions
2. Configure voicemail
3. Set up auto-attendant (if included)
4. Configure call forwarding rules
5. Test inbound/outbound calling

**Step 4: Training (15-30 minutes)**
1. Demonstrate phone features
2. Set up voicemail greetings
3. Show mobile app installation
4. Review admin portal
5. Provide quick reference guide

### Managed WiFi Installation Procedure

**Duration**: 3-6 hours

**Step 1: Site Survey (if not already completed)**
1. Walk the premises
2. Identify optimal AP locations
3. Check for interference
4. Plan cable runs

**Step 2: Access Point Installation (2-3 hours)**
1. Mount access points at planned locations
2. Run Ethernet cables from router/switch
3. Install PoE (Power over Ethernet) injectors if needed
4. Power on access points
5. Verify network connectivity

**Step 3: Controller Configuration (1-2 hours)**
1. Install and configure network controller
2. Adopt access points
3. Configure SSIDs (primary and guest networks)
4. Set up security (WPA2/WPA3)
5. Configure guest portal (if required)
6. Optimize channel selection
7. Set transmit power levels

**Step 4: Testing and Optimization (30-60 minutes)**
1. Perform site walk with WiFi analyzer
2. Test coverage in all areas
3. Verify minimum signal strength (-67 dBm)
4. Test client connectivity
5. Verify guest network isolation
6. Run speed tests in multiple locations

**Step 5: Documentation and Training (30 minutes)**
1. Create WiFi coverage map
2. Document SSID and passwords
3. Train customer on admin portal
4. Provide guest network management instructions

### Managed Security Installation Procedure

**Duration**: 2-3 hours

**Step 1: Firewall Installation (30-45 minutes)**
1. Unbox and inventory equipment
2. Install firewall between router and network
3. Connect WAN port to router
4. Connect LAN port to customer network
5. Power on firewall

**Step 2: Initial Configuration (60-90 minutes)**
1. Access firewall admin interface
2. Configure WAN settings
3. Configure LAN settings
4. Set up firewall rules
5. Enable intrusion prevention
6. Configure content filtering
7. Set up VPN (if required)

**Step 3: Security Policy Configuration (30-45 minutes)**
1. Define security zones
2. Configure application control
3. Set up web filtering categories
4. Enable antivirus scanning
5. Configure logging and alerts

**Step 4: Testing (30 minutes)**
1. Test internet connectivity
2. Verify firewall rules
3. Test blocked content
4. Verify VPN connectivity (if configured)
5. Generate test security events

**Step 5: Training and Documentation (15-30 minutes)**
1. Review security dashboard
2. Explain alert notifications
3. Demonstrate report access
4. Provide security best practices guide

## Troubleshooting Common Installation Issues

### Fiber Signal Issues
**Symptom**: No link light on ONT
**Causes**:
- Fiber not properly connected
- Fiber damaged during installation
- Incorrect fiber type

**Resolution**:
1. Check fiber connections at ONT and demarcation point
2. Inspect fiber for damage (kinks, cuts)
3. Test fiber with OTDR (Optical Time Domain Reflectometer)
4. Replace fiber if damaged
5. Escalate to outside plant team if issue persists

### Router Configuration Issues
**Symptom**: Internet not working after installation
**Causes**:
- Incorrect WAN configuration
- IP conflict
- Firewall blocking traffic

**Resolution**:
1. Verify WAN settings (DHCP or static IP)
2. Check for IP conflicts
3. Review firewall rules
4. Reset router to factory defaults and reconfigure
5. Contact network operations center for assistance

### WiFi Coverage Issues
**Symptom**: Weak signal in some areas
**Causes**:
- Insufficient access points
- Poor AP placement
- Interference from other devices

**Resolution**:
1. Perform site survey with WiFi analyzer
2. Identify dead zones
3. Adjust AP placement or add additional APs
4. Change channels to avoid interference
5. Adjust transmit power levels

### Voice Quality Issues
**Symptom**: Poor call quality, dropped calls
**Causes**:
- Insufficient bandwidth
- Network congestion
- QoS not configured

**Resolution**:
1. Verify internet bandwidth adequate for voice
2. Configure QoS on router
3. Set up voice VLAN
4. Test network latency and jitter
5. Escalate to voice engineering team if needed

## Equipment Return and Swap Procedures

### Equipment Swap (Defective Equipment)
**Process**:
1. Customer reports equipment issue
2. Support troubleshoots remotely
3. If defective, replacement equipment shipped overnight
4. Customer swaps equipment (self-install) or technician dispatched
5. Defective equipment returned via prepaid label
6. No charge to customer

### Service Downgrade Equipment Return
**Process**:
1. Customer downgrades service
2. Equipment return kit shipped to customer
3. Customer has 30 days to return equipment
4. Equipment received and inspected
5. If not returned or damaged, customer charged

### Service Cancellation Equipment Return
**Process**:
1. Service cancelled
2. Equipment return kit shipped within 3 business days
3. Customer has 30 days to return all equipment
4. Equipment received and inspected
5. Unreturned equipment: $500 charge per router, $150 per phone
6. Damaged equipment: Repair cost charged to customer

## Emergency and After-Hours Installations

### Emergency Installation Criteria
- Service outage at existing customer

1. Customer requests weekend installation
